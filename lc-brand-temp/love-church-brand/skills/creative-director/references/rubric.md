# Rubric: 1–10 anchors per category

Use these anchors every time you score. Without them, scores drift toward the middle and the rubric becomes useless.

**Calibration mantra:** 5 is functional. 7 is solid. 9 is exceptional. 10 is inevitable. The gap between a 7 and a 9 is bigger than the gap between a 3 and a 7. Most professional work lives between 5 and 7; that is not a failure, that is the truth about most work.

A few rules that apply to every category:

- If you find yourself giving 6/7 across the board, stop. The scores should *vary*. Variance is information.
- 10/10 means you couldn't make it better. Reserve it. A team that earns a 10 once a year is doing extraordinary work.
- When in doubt between two scores, pick the lower one. It's more useful for coaching.

---

## Composition

*"Is this built, or is this floating?"*

| Score | What it looks like |
|---|---|
| **1–2** | No structure. Random placement, no alignment, no grid. Elements touch edges by accident. White space is "leftover" rather than designed. |
| **3–4** | Visible attempt at a layout but multiple alignment errors. Awkward gaps. Elements crowd one corner while another corner sits empty for no reason. |
| **5** | Functional. Centered stack or basic two-column. Nothing is broken but nothing is built either. The composition is "default." |
| **6–7** | Disciplined. A grid is visible underneath. Margins are intentional. Negative space is being used as a tool, not as filler. |
| **8** | Considered. Asymmetry feels earned. Edge tension is doing work. The composition has rhythm — a measurable beat between elements. |
| **9** | Rare. The composition itself carries meaning before you read a word. You sense the structure as quickly as you see the image. |
| **10** | Inevitable. Move any element and the piece gets worse. The grid disappears because everything sits exactly where it must. |

## Color

*"Could you remove a color and the piece would be stronger?"*

| Score | What it looks like |
|---|---|
| **1–2** | Random palette. Illegible contrast. Saturated colors fighting each other. Brand colors thrown in as defaults with no thought. |
| **3–4** | Too many colors. Or two saturated colors next to each other for no reason. Color is decoration, not communication. |
| **5** | Safe palette. Two or three colors that don't fight. Reads OK but doesn't carry meaning. |
| **6–7** | Restrained. Every color in the piece is doing intentional work. Contrast is sufficient (and likely WCAG AA compliant). |
| **8** | One dominant color carrying the mood, with one supporting accent. Or an honest duotone that earns its restraint. |
| **9** | The color *is* the idea. You couldn't tell the story in a different palette. |
| **10** | Color and meaning are inseparable. Black and white at this score is as much a choice as scarlet. |

## Typography

*"Is the typography invisible because it's right, or invisible because it's lazy?"*

| Score | What it looks like |
|---|---|
| **1–2** | Disqualifying font choices (Comic Sans, Papyrus, default decorative fonts). Multiple sizes in one line. All caps with drop shadow. Stretching or distorting type. |
| **3–4** | Free fonts pretending to be Helvetica. Awkward pairings (two display faces, a serif and a script that hate each other). Cliché choices used as default (Bebas Neue, Montserrat ExtraBold) without intention. |
| **5** | One sensible typeface. Sizing in the right ballpark. Reads clearly. Forgettable. |
| **6–7** | Two faces, deliberately paired. Scale steps are intentional (not 18, 22, 24, 30 by accident). Tracking and leading have been considered. |
| **8** | One typeface across three weights doing all the work. The discipline is visible. Or a pairing where each face is doing a job the other can't. |
| **9** | The typography itself communicates the tone of the piece — austere, warm, urgent — before any image does. |
| **10** | The piece would still work if you removed everything but the type. The type is the design. |

## Hierarchy

*"Cover the headline. Can you still tell what this is about?"* If yes, hierarchy is broken.

| Score | What it looks like |
|---|---|
| **1–2** | No hierarchy. Every element is equally loud, or equally quiet. The eye doesn't know where to start. |
| **3–4** | Two equal-weight elements compete. Or one element is so loud (oversized headline, blown-out photo) that everything else is wasted. |
| **5** | First read is clear. Second and third reads are muddled. The piece dies after the headline. |
| **6–7** | Clear 1-2-3 read. Headline. Subhead. Detail. Eye knows what to do. |
| **8** | The hierarchy *is* the design. Removing any one element would break the order, not just weaken it. |
| **9** | You see the message before you read the words. Pre-attention is doing the work of attention. |
| **10** | Hierarchy invisible because it's right. The piece reads in the only order it could read in. |

## Storytelling

*"In one sentence, what is this about?"* If you can't say it, the work isn't.

| Score | What it looks like |
|---|---|
| **1–2** | No idea. Decoration only. The piece is wallpaper. |
| **3–4** | An idea is in there but buried under decoration, copy, or competing visuals. |
| **5** | One idea, plainly stated. Not memorable. Functional message delivery. |
| **6–7** | One idea, well-said. Tone matches subject. The piece earns attention. |
| **8** | One idea, said in a way only this piece could say it. The visual angle is specific. |
| **9** | The piece *is* the idea. You'd remember it tomorrow without effort. |
| **10** | You'd remember it next year. The piece becomes part of how the audience thinks about the subject. |

## Emotional Impact

*"Would this make a stranger stop scrolling?"* — answer that without flattering the work.

Before scoring, name the **target emotion** (awe, conviction, invitation, urgency, comfort, joy, sobriety). You can't grade emotional impact without naming the target first.

| Score | What it looks like |
|---|---|
| **1–2** | Nothing. Or worse — the wrong feeling for the subject (a goofy headline on a piece about grief). |
| **3–4** | Generic feeling. "Inspirational" without specificity. The piece could be advertising anything. |
| **5** | Reads as competent. Doesn't move you. You'd scroll past it. |
| **6–7** | Hits the intended note. You feel it on a second look, after you've parsed the message. |
| **8** | Earns the feeling — you feel it *before* you understand why. The image works on you faster than the headline does. |
| **9** | The piece changes how you feel about its subject for that moment. Rare. |
| **10** | You can't unsee it. Days later, the feeling is still attached to the subject. |

## How to handle pieces that don't have all six dimensions

Some pieces will score lower in a category not because they're weak, but because the category is barely involved. A 1:1 photo-only social post has minimal typography. A motion intro card may have minimal storytelling on its own.

In those cases, score honestly against the dimension *as it appears in the piece*. Don't grade out of pity. If typography is barely present and what's there is a weak treatment, score the typography weak. Note the situation in the "Why It Works / Why It Doesn't" paragraph: *"Typography is barely present in this piece, so its weakness is a small drag on the total. The composition is doing nearly all the work."*
